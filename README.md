<div align="center">
  <h2>📱 Usage</h2>
</div>

<div align="center">
  <p>The extension works automatically when you visit the LPU internet login pages:</p>
  <code>https://internet.lpu.in/24online/servlet/E24onlineHTTPClient</code><br>
  <code>https://internet.lpu.in/24online/webpages/client.jsp</code>
</div>

<table align="center" width="100%" style="margin-top: 20px;">
  <tr style="background-color: #f5f5f5;">
    <th width="15%" align="center">Step</th>
    <th width="40%" align="center">Your Action</th>
    <th width="45%" align="center">Extension Response</th>
  </tr>
  <tr align="center">
    <td><b>1</b></td>
    <td>Visit LPU Internet login page</td>
    <td>Extension activates automatically</td>
  </tr>
  <tr align="center">
    <td><b>2</b></td>
    <td>Enter your username (Registration Number)</td>
    <td>Removes "@lpu.com" if you accidentally type it</td>
  </tr>
  <tr align="center">
    <td><b>3</b></td>
    <td>Enter your password</td>
    <td>Automatically enables login button</td>
  </tr>
  <tr align="center">
    <td><b>4</b></td>
    <td>Nothing! Just wait</td>
    <td>Extension clicks login button for you</td>
  </tr>
</table>
