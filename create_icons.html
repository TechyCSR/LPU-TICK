PNG Icon Creator for LPU Internet Login Helper

This page will automatically convert SVG icons to PNG format for use in the Chrome Extension.

When the conversion is complete, you'll be prompted to download the PNG files.

Instructions:

  1. Save the downloaded PNG files to the icons/ folder of your extension.
  2. Update the manifest.json to use these PNG files instead of SVG files.
  3. These PNG icons will be used for the Chrome Web Store and extension UI.

Note: SVG files may not display correctly in the Chrome Web Store, so PNG files are recommended.
